ticketmaster by

Ticketmaster provides a universal API to trouble ticket and project management systems.

Dependencies

Very few.

Install

Fire up terminal 

    gem install ticketmaster

then just install whatever providers you want to use, for example: 

    gem install ticketmaster-pivotal
